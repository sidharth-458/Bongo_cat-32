# Bongo_cat-32
Controlling Bongo cat through esp32 and i2c oled display.\
Items needed:
- bunch of jumper cables
- an ESP-32
- aduino ide setup for esp32
- i2c oled board 128x64

Connections:-\
ESP-32-------OLED\
3.3----------VCC\
GND----------GND\
22-----------SCL\
21-----------SDA

Use esp_32 capacitive touch pins at  I04/D4 and IO15/D15
